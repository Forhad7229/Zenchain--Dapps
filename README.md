# Zenchain DApp Demo

This is a simple decentralized application (DApp) demo built for the Zenchain testnet.  
It includes a basic ERC20 token smart contract written in Solidity and deployed using Hardhat.

## 🛠️ Tools Used
- Solidity
- Hardhat
- Ethers.js

## 🚀 How to Run

```bash
npm install
npx hardhat compile
npx hardhat test
